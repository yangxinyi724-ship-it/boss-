## RAG 空参考展示准确原因

## Embedding / 硅基流动接线与回填

## 公开发布评估
- **类型**：说明 / 文档

## SSE 积压丢岗位 / 监控无浏览细节
- **类型**：修复
- **问题**：页数涨无岗位；监控翻页直接到页完成；最新进展被警告盖住
- **原因**：队列满丢最旧事件（含 `job_passed`）；曾停推浏览事件
- **处理**：关键事件优先；可丢浏览；live 快照补拉；恢复必要浏览推送

## 页面隐藏误停 / 心跳与续跑策略
- **类型**：修复 / 改动
- **问题**：最小化约 90s 被停；或 Web 卡、终端仍跑
- **原因**：后台节流 ack；生命周期绑在 Web
- **处理**：`hidden` + Worker 心跳；放宽超时；**最终隐藏仍续跑**，回前台同步 UI

## 轮次休息被前端误唤醒
- **类型**：修复
- **问题**：后端休息，UI/工位像在搜
- **原因**：ring/页码误触发 resume；隐藏文案进标题
- **处理**：收紧唤醒；标题以统计为准，隐藏提示进监控动作

## 下班暂停被 watchdog 误杀
- **类型**：修复
- **原因**：下班事件未进豁免；长静默无心跳
- **处理**：休息/下班豁免无进展判定

## pet.js 括号写坏导致整页卡「加载中」
- **类型**：修复
- **问题**：脚本解析失败，监控「未连接」

# 跨阶段反复出现的主题

| 主题 | 本质 |
|------|------|
| 终端 vs Web 不同步 | SSE 丢事件、后台节流、Chromium 崩、Playwright 线程、快照不全 |
| 隐藏/关页影响搜岗 | 生命周期曾绑 Web；目标改为后端持有任务、Web 只订阅 |
| stoken / 深分页 | 约 5 次失效、刷新、失败重开浏览器、勿回首页死循环 |
| 休息/作息状态机 | 疲劳、轮次休息、下班、误唤醒、误杀 |
| 浏览器线程模型 | greenlet、单线程 executor、勿把 SQLite/整段 pipeline 塞进浏览器线程 |
| 换词策略 | 深度达标 → 列表扫完；取消按通过数 |
| RAG | 关键词 → 向量；配置接线；职业阶段漏接；空结果文案 |
| 资料柜 | 拒绝学习 → 通过历史 → 不感兴趣 → BOSS 查看 |
