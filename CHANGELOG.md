# Changelog

## 1.11.0

### 主界面

- 以宠物页 `/pet` 为默认主界面（`/` 重定向到 `/pet`）
- AI 办公室多工位：侦察 / 分析 / 监控 / 秘书 + 资料柜、通过岗位栏
- 静态资源按文件 mtime 自动缓存版本号

### 搜岗

- 后端持有搜岗生命周期；Web 仅订阅 SSE（刷新/断开不断任务）
- 省 → 市 → 区三级地区；列表扫完再换词（不按通过岗数换词）
- 工作时段自动开搜 / 下班暂停；下班暂停不再被 watchdog 误判为卡死
- SSE 优先保留岗位通过事件；浏览/跳过信息可显示，积压时可丢
- live 快照可补拉已通过岗位与统计

### 其它

- Playwright / patchright 浏览器操作固定在专用线程，避免 asyncio 循环报错
- 标题小字展示搜岗统计：已侦察 / 侦察通过 / 分析通过

### 包信息

- CLI 入口：`boss`
- Python 包：`pet_boss`
- PyPI 工程名：`boss-agent-cli`
